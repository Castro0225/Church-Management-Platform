# ChurchFlow

ChurchFlow is a Church Management Platform designed to help churches organize their members, finances, events, and ministry schedules in a simple and transparent way.

## Features

- Member management
- Financial management (tithes, offerings, and expenses)
- Ministry organization
- Event management
- Volunteer scheduling
- Reports and dashboards

## Tech Stack

Backend:
- Java
- Spring Boot

Frontend:
- React / Next.js

Database:
- PostgreSQL

## Project Status

🚧 In development

## Goal

Create a modern SaaS ERP platform to help churches manage their administration and improve transparency and organization.
